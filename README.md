# Wushu-Arena-Car
这是一个基于ESP32-S3开发的武术擂台车控制系统，支持自动寻边、避障、推棋子等功能,曾在华北五省机器人大赛中获得一等奖，关于机械结构，采用了双轮驱动结构，红外传感器分布在车的两侧，用于检测障碍物和棋子，电机驱动模块用于控制车的运动，光电传感器用于检测车的位置，MPU6050陀螺仪模块用于检测车的姿态，OLED显示屏用于显示车的状态和运行信息。
